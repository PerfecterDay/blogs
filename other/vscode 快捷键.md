
# VS code 快捷键
{docsify-updated}

+ 上一行或下一行添加光标（多行编辑）：`Ctrl+Alt+DownArrow/UpArrow`
+ 任意处添加光标：`Alt+Click`
+ 选中所有当前查找到的文本：`Ctrl+Shift+L`
    macos下的操作步骤：
    1. 按下 Cmd + F（⌘F） 打开查找栏
    2. 输入你要查找的文本
    3. 按下 Alt + Enter（⌥ + ⏎）
+ 复制行：`Shift+Alt+DownArrow/UpArrow`
+ 移动行：`Alt+UpArrow/DownArrow`
+ 删除行：`Ctrl+Shift+K`
+ 格式化整个文档: `Shift+Alt+F`
+ 格式化选中文本： `Ctrl+K Ctrl+F`
+ 折叠： `Ctrl+Shift+[`
